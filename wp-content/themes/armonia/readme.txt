== Description ==
Armonia is a responsive WordPress theme that lets you write blog posts and articles easily. It has very easy-to-use customizer options. It comes with multiple blog posts layouts and useful widgets that completes your blog website requirements. It provides dozens of amazing features like numerous colors option, site title typography with more than 600+ google fonts, sticky header, social icons, button layouts, page breadcrumbs, sidebar layouts, sticky sidebar, scroll to top options, and much more to explore inside the theme. We provide free support for you and are glad to help you out with any type of theme issue. It uses the best clean SEO practices. Built more than 50+ layouts with possible combinations of blocks available. Check out our beautiful pre-built demos now: https://blazethemes.com/theme/armonia?section=themesingle_demo_wrap

== Installation ==
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Search for armonia theme. Click Install Now.
3. Click Upload Theme and Choose File, then select the armonia.zip file. Click Install Now.
4. Click Activate to use your new theme right away.
