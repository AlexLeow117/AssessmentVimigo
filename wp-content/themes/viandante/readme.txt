== Description ==

Viandante is a clean Wordpress theme, perfect to create a personal blog or and magazine website. Viandante is easily customizable thanks to the WordPress Live Customizer section.

Developed by ThemeinProgress, https://www.themeinprogress.com

Viandante WordPress Theme is a child theme of Avventura Lite Theme, Copyright 2022 ThemeinProgress.
Viandante is distributed under the terms of the GNU GPLv3.
